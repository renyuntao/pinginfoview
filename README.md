## DESCRIPTION

PingInfoView is a small utility that allows you to easily ping multiple host names and IP addresses, and watch the result in one table.

## USAGE

```bash
$ ./pinginfoview -h 

Usage:   
        ./pinginfoview FILENAME   
        ./pinginfoview -h   

Interactive Commands:   
        n       Show next page   
        q       Quit   
```

## EXAMPLE

$ ./pinginfoview ./file
